# 使用者角色 (Personas)

本平台有三種主要的使用者角色，其權限與需求各不相同。

## 1. 一般訪客 (Visitor)
- **描述**: 對綠色生活感興趣的潛在使用者。
- **需求**: 快速查找。例如：「我現在在高雄火車站附近，哪裡有提供自備杯優惠的咖啡廳？」或「我想找一間『全素』餐廳。」
- **核心功能**:
  - 瀏覽與操作互動式地圖 (F-A1)
  - 點擊圖釘查看詳情面板 (F-A3)
  - 使用關鍵字搜尋 (F-A4)
  - 使用分類與「綠活標籤」進行篩選 (F-A5)
- **權限**: 唯讀 (Read-only)。

## 2. 綠活伙伴 (Partner)
- **描述**: 熱心的社群貢獻者，是平台活化的關鍵。
- **需求**: 貢獻資料。例如：「我發現這家麵店內用都不提供免洗餐具，我要把它分享上去！」或「這家店倒了，我要回報。」
- **核心功能**:
  - 包含「一般訪客」所有功能。
  - 透過社交帳號快速登入 (F-B1 Auth)。
  - 透過「快速上傳」表單提交新地點 (F-B1)。
  - 上傳照片 (F-B1 Photos)。
  - 對現有地點「回報錯誤」 (F-B3)。
- **權限**: 需登入。可讀取 (Read) 並建立 (Create) 新資料（需經審核）。

## 3. 平台管理員 (Admin)
- **描述**: 平台的核心維護者，確保資料品質。
- **需求**: 維護資料庫的準確性與品質。
- **核心功能**:
  - 審核「待審核」的新地點 (F-C1)。
  - 管理（新增/修改/刪除）所有地點、標籤 (F-C2, F-C3)。
  - 處理來自伙伴的「待處理回報」清單 (F-C2)。
- **權限**: 需特殊權限登入。擁有完整的資料庫讀寫權限。
