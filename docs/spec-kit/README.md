# 專案：南區綠活圖資訊平台 (v1.0) - 規格書 (Spec-Kit)

## 專案總覽

本專案旨在建立一個由社群驅動、地圖導向的綠色生活資訊共享平台。透過「伙伴」的即時貢獻，讓使用者能快速查找（Search）並簡易上傳（Upload）符合綠色生活標準的店家、地點與活動場域。

本規格書（spec-kit）旨在為 AI 開發者提供清晰、可執行的開發藍圖。

## 規格文件結構

本專案的完整規格被拆分至以下文件：

- **[PROJECT_BRIEF.md](./PROJECT_BRIEF.md)**: 專案願景、目標與市場機會。
- **[PERSONAS.md](./PERSONAS.md)**: 使用者角色定義。
- **[FUNCTIONAL_SPECS.md](./FUNCTIONAL_SPECS.md)**: 完整的功能性規格（A, B, C 模組）。
- **[DATA_MODEL.md](./DATA_MODEL.md)**: 資料庫結構。
- **[TECH_STACK.md](./TECH_STACK.md)**: 技術棧與架構決策。
- **[SECURITY_SPECS.md](./SECURITY_SPECS.md)**: 資訊安全規格。
- **[DEVELOPMENT_GUIDE.md](./DEVELOPMENT_GUIDE.md)**: 開發環境設定、初始化與管理員設定指南。
