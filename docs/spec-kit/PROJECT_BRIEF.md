# 專案簡報 (Project Brief)

## 1. 專案願景 (Vision)
建立一個由社群驅動、地圖導向的綠色生活資訊共享平台。讓「綠色生活」在台灣（初期由南區開始）變得更便利、更可見，並透過「伙伴」的貢獻力量，活化整個永續生活社群。

## 2. 核心目標 (Goals)
- **快速查找 (Search)**: 使用者能透過地圖和精確的「綠活標籤」（例如：#全素/蔬食店），在 30 秒內找到符合需求的店家。
- **簡易貢獻 (Upload)**: 「伙伴」能透過手機，在 1 分鐘內快速上傳新的地點或回報現有資訊，確保平台的即時性與活躍度。
- **社群活化 (Activate)**: 透過明確的貢獻機制與回報系統，建立一個可自我維護與更新的資料庫。

## 3. 市場定位與機會點 (Opportunity)
經分析，現有平台多為「靜態地圖」（部落格整理）或「狹窄地圖」（僅特定主題或品牌）。本平台的核心價值 (Value Proposition) 在於：
- **「廣度整合」**: 整合「食、衣、住、行、育、樂」外加「團體活動場域」，滿足多元需求。
- **「即時貢獻」**: 打造一個極簡便的「伙伴上傳」(F-B1) 流程，解決現有平台資料老舊的痛點。
- **「精確標籤」**: 提供如「全素 v.s. 提供素食」的精確圖標 (Core Icons)，解決使用者搜尋時的困擾。
