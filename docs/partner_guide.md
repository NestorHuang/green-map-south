# 綠活伙伴操作手冊 (Partner Guide)

歡迎成為「南區綠活圖」的綠活伙伴！您的貢獻是平台成長的關鍵。本手冊將引導您如何使用伙伴專屬的功能。

**作為伙伴，您擁有一般訪客的所有功能。** 請先參考 `visitor_guide.md` 了解基本的地圖操作、搜尋與篩選功能。

本手冊將著重於伙伴的兩大核心功能：**上傳新地點**與**回報錯誤**。

## 1. 登入您的帳號

要使用伙伴功能，您必須先登入。

- **點擊登入**：在地圖頁面的右上角，點擊「使用 Google 登入」按鈕。
- **授權**：在彈出的視窗中，選擇您的 Google 帳號並完成授權。
- **登入成功**：登入後，原本的登入按鈕會變為您的使用者名稱、一個「上傳地點」的連結以及「登出」按鈕。

## 2. 上傳新的綠活地點 (F-B1)

當您發現一個很棒但地圖上還沒有的綠活地點時，您可以將它分享出來！

1.  **前往上傳頁面**：登入後，點擊右上角的「上傳地點」連結，即可進入上傳表單頁面。

2.  **填寫表單**：
    - **地點名稱***：必填，請填寫店家或地點的完整名稱。
    - **地址***：必填，請填寫完整的地址。系統會根據此地址自動轉換為地圖座標。
    - **描述**：選填，您可以分享這個地點的特色或值得推薦之處。
    - **綠活標籤**：從列表中勾選所有符合此地點的標籤。
    - **照片***：必填，請上傳一張能代表此地點的照片。

3.  **提交審核**：
    - 填寫完畢後，點擊「提交審核」按鈕。
    - 系統會處理您的資料，成功後會提示您「地點已成功提交審核！」，並自動將您導回地圖首頁。
    - 您提交的地點將會進入管理員的審核佇列，審核通過後就會出現在公開地圖上。

## 3. 回報現有地點的錯誤 (F-B3)

如果發現某個地點的資訊有誤（例如：已歇業、地址錯誤、標籤不符），您可以協助回報。

1.  **開啟地點資訊面板**：在地圖上點擊您想回報的地點圖釘。
2.  **點擊回報按鈕**：在資訊面板的下方，您會看到一個紅色的「回報此地點資訊有誤」按鈕，點擊它。
3.  **填寫回報內容**：
    - 畫面上會彈出一個回報視窗。
    - 在文字框中，請盡可能詳細地描述您發現的問題。
    - 點擊「送出回報」。
4.  **完成**：您的回報將會送至管理員的處理佇列中。
