## 快速入門

### 什麼是綠地圖南方？

綠地圖南方是荒野保護協會南方分會的地圖平台，讓您可以：
- 🗺️ **探索** 南台灣的綠色空間與自然教育場域
- 📍 **分享** 您知道的特色地點
- 📝 **記錄** 場域的詳細資訊與使用經驗
- 💬 **交流** 與其他荒野夥伴的活動心得

### 訪客 vs 登入使用者

| 功能 | 訪客（未登入） | 登入使用者 |
|------|--------------|-----------|
| 瀏覽地圖 | ✅ | ✅ |
| 查看地點詳情 | ✅ | ✅ |
| 提交新地點 | ❌ | ✅ |
| 編輯地點 | ❌ | ✅（僅自己的） |
| 儲存草稿 | ❌ | ✅ |

### 首頁地圖

進入網站後，您會看到主地圖頁面：

![首頁地圖](images/01-homepage-map.png)

#### 地圖操作

**🖱️ 基本操作**

```
┌─────────────────────────────────┐
│  🔍 搜尋地址    🏷️ 篩選標籤    │  ← 頂部工具列
├─────────────────────────────────┤
│         🗺️ 地圖區域              │
│                                 │
│    📍 📍   📍                    │  ← 地點標記
│  📍      📍  📍                  │
│                                 │
│    [➕] [➖] [📍]                │  ← 地圖控制
└─────────────────────────────────┘
```

1. **移動地圖**
   - 電腦：按住滑鼠左鍵拖曳
   - 手機：手指滑動

2. **縮放地圖**
   - 電腦：滑鼠滾輪 或 點擊 ➕➖ 按鈕
   - 手機：雙指縮放 或 點擊 ➕➖ 按鈕

3. **回到我的位置**
   - 點擊右下角的 📍 定位按鈕
   - 首次使用需允許瀏覽器取得位置權限

![地圖控制](images/02-map-controls.png)
> *📸 截圖說明：地圖右下角的縮放和定位按鈕*

### 搜尋地址

想要查看特定區域的地點？使用搜尋功能：

1. 點擊頂部的 **🔍 搜尋地址** 輸入框
2. 輸入地址或地標名稱（例如：「高雄市三民區」）
3. 從下拉選單選擇建議的地址
4. 地圖會自動移動到該位置

![搜尋地址](images/03-search-address.png)
> *📸 截圖說明：搜尋框輸入文字後顯示建議地址的下拉選單*

### 篩選標籤

想只看特定類型的地點？使用標籤篩選：

1. 點擊頂部的 **🏷️ 篩選標籤** 按鈕
2. 選擇您想查看的標籤（例如：「戶外」、「室內」）
3. 地圖上會只顯示符合該標籤的地點
4. 再次點擊標籤或點擊「清除」取消篩選

![標籤篩選](images/04-tag-filter.png)
> *📸 截圖說明：點擊標籤按鈕後展開的標籤選單*

### 地點標記圖示

地圖上的標記會依照地點類型顯示不同的圖示和顏色：

```
🏞️  - 戶外場域（綠色）
🏛️  - 室內場域（藍色）
🌳  - 公園綠地（淺綠色）
🏫  - 教育機構（橙色）
```

- ✅ 編輯自己提交的地點
- ✅ 儲存草稿，稍後繼續編輯
- ✅ 回報地點資訊有誤

1. 點擊右上角的 **「登入」** 按鈕

## 提交新地點

### 開始提交

**方式一：從地圖新增**

1. 在地圖上找到您想標記的位置
2. 長按地圖該位置（手機）或點擊右鍵（電腦）
3. 選擇「在此新增地點」

![地圖新增](images/16-add-from-map.png)
> *📸 截圖說明：在地圖上長按顯示的選單*

**方式二：從選單新增**

1. 點擊右上角個人選單
2. 選擇「➕ 新增地點」

![選單新增](images/17-add-from-menu.png)
> *📸 截圖說明：個人選單中的「新增地點」選項*

### 儲存草稿

**不確定要立即提交？**先儲存為草稿：

1. 填寫部分資訊
2. 點擊 **「💾 儲存草稿」** 按鈕
3. 稍後可從「我的草稿」繼續編輯

3. 等待管理員審核（通常 1-3 天）

## 編輯我的地點

### 查看我提交的地點

1. 點擊右上角頭像
2. 選擇 **「📝 我的地點」**

![我的地點選單](images/29-my-locations-menu.png)
> *📸 截圖說明：個人選單中的「我的地點」選項*

### 我的地點列表

![我的地點列表](images/30-my-locations-list.png)
> *📸 截圖說明：我的地點列表頁面*

顯示您提交的所有地點及其狀態：

```
┌─────────────────────────────────┐
│  📝 我的地點                     │
├─────────────────────────────────┤
│  ✅ 中都愛河濕地公園             │
│     狀態：已核准                │
│     [查看]  [編輯]               │
├─────────────────────────────────┤
│  ⏳ 原住民主題公園               │
│     狀態：審核中                │
│     [查看]                      │
├─────────────────────────────────┤
│  ❌ 某某地點                    │
│     狀態：已退回                │
│     原因：地址資訊不完整         │
│     [修改後重新提交]             │
└─────────────────────────────────┘
```

### 狀態說明

- ✅ **已核准**：地點已通過審核，在地圖上顯示
- ⏳ **審核中**：等待管理員審核
- ❌ **已退回**：審核未通過，可修改後重新提交
- 🔒 **已鎖定**：管理員鎖定，無法編輯

### 編輯已核准的地點

1. 在「我的地點」中找到要編輯的地點
2. 點擊 **「✏️ 編輯」** 按鈕
3. 修改需要更新的資訊
4. 點擊 **「💾 儲存變更」**

![編輯地點](images/31-edit-location.png)
> *📸 截圖說明：地點編輯表單*

**⚠️ 注意事項：**
- 只能編輯自己提交的地點
- 被管理員鎖定的地點無法編輯
- 重大變更可能需要重新審核

## 常見問題

### ❓ 我的地點提交後多久會顯示？

通常管理員會在 **1-3 個工作天**內完成審核。審核通過後地點會立即顯示在地圖上。

### ❓ 為什麼我的地點被退回？

常見退回原因：
- 📍 地點位置不正確
- 📝 資訊不完整或不清楚
- 📷 照片品質不佳或與地點無關
- 🔁 重複提交已存在的地點

收到退回通知時，請查看退回原因，修改後重新提交。

### ❓ 可以刪除我提交的地點嗎？

已核准的地點**無法自行刪除**，請聯繫管理員協助處理。
草稿可以隨時刪除。

### ❓ 忘記密碼怎麼辦？

我們使用 Google 登入，請：
1. 前往 Google 帳號設定
2. 重設 Google 帳號密碼
3. 使用新密碼重新登入綠地圖

### ❓ 可以修改其他人提交的地點嗎？

不行，一般使用者**只能編輯自己提交的地點**。
如果發現其他地點資訊有誤，請使用「🚨 回報此地點資訊有誤」功能。

### ❓ 如何回報地點資訊有誤？

1. 開啟該地點的詳情頁面
2. 捲動到底部
3. 點擊 **「🚨 回報此地點資訊有誤」**
4. 填寫問題描述
5. 送出回報

![回報按鈕](images/32-report-button.png)
> *📸 截圖說明：地點詳情頁底部的回報按鈕*

### ❓ 我可以上傳幾張照片？

每個地點最多可上傳 **10 張照片**。

建議上傳：
- 地點全景照
- 入口或標示牌
- 特色景觀
- 設施照片

### ❓ 手機版和電腦版功能一樣嗎？

是的！所有功能在手機和電腦上都可以使用，只是介面會針對螢幕大小調整。

### ❓ 需要下載 APP 嗎？

不需要！綠地圖是**網頁版應用**，使用瀏覽器即可。

建議的瀏覽器：
- 💻 電腦：Chrome、Firefox、Edge、Safari
- 📱 手機：Chrome、Safari

### ❓ 可以離線使用嗎？

目前**不支援離線使用**，需要網路連線才能查看地圖和提交資料。

## 💡 使用技巧

### 技巧 1：善用標籤篩選

計劃活動前，使用標籤快速找到符合需求的場域。
例如：篩選「戶外」+「親子」找適合家庭活動的地點。

### 技巧 2：先儲存草稿

外出探勘時，可以先用手機快速記錄地點資訊，儲存為草稿，
回家後再用電腦補充完整內容和上傳照片。

### 技巧 3：詳細描述更有價值

提交地點時，請盡量填寫詳細資訊：
- ✅ 容納人數、場地大小
- ✅ 交通方式、停車資訊
- ✅ 使用限制、注意事項
- ✅ 最佳使用時段

### 技巧 4：照片要清晰完整

- 📷 白天光線充足時拍攝
- 📷 包含全景和細節照片
- 📷 拍攝入口或標示方便找到
- 📷 展現場地特色

### 技巧 5：定期更新資訊

如果您發現自己提交的地點資訊有變動（例如：聯絡電話更改、
設施整修），請記得更新，讓其他夥伴獲得最新資訊。
