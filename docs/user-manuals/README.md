## 🎯 使用建議

### 給一般使用者

📖 **推薦閱讀順序**：

1. **新手上路**
   - 從 [使用者操作手冊](user_operation_guide.md) 的「快速入門」開始
   - 了解基本的地圖瀏覽功能

2. **進階使用**
   - 學習如何登入
   - 練習提交新地點
   - 使用草稿功能

3. **高級功能**
   - 編輯已提交的地點
   - 回報問題

### 給內容創作者

如果您要為手冊補充截圖：

1. 閱讀 [截圖指引](SCREENSHOT_GUIDE.md)
2. 準備截圖工具
3. 按照優先順序拍攝
4. 存放至 `images/` 目錄
5. 確保檔名正確

## 📝 維護說明

### 更新手冊

當系統功能更新時，請：

1. 更新 [使用者操作手冊](user_operation_guide.md) 內容
2. 補充或替換相關截圖
3. 更新 [截圖指引](SCREENSHOT_GUIDE.md)（如有新增截圖）
4. 更新本 README 的最後更新日期

### 翻譯

目前所有手冊均為繁體中文。
如需其他語言版本，請在同目錄下創建：
- `user_operation_guide_en.md`（英文版）
- `user_operation_guide_jp.md`（日文版）
