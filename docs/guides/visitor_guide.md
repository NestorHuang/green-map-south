# 一般訪客操作手冊 (Visitor Guide)

歡迎使用「南區綠活圖」！本平台旨在幫助您輕鬆快速地找到符合綠色生活標準的地點。作為一般訪客，您可以存取所有公開的搜尋與瀏覽功能。

## 1. 瀏覽互動地圖

開啟平台時，您會看到一個以您目前位置為中心的地圖（如果您同意分享位置資訊）。

- **移動地圖**：按住滑鼠左鍵拖曳即可平移地圖。
- **縮放地圖**：使用滑鼠滾輪或地圖上的 `+` / `-` 按鈕來縮放。

## 2. 搜尋與篩選

地圖頂端是我們的搜尋與篩選工具列，幫助您精準找到目標。

### 關鍵字搜尋

在搜尋框中輸入您想找的地點名稱（或名稱的一部分），地圖上的標記將會即時更新，只顯示符合條件的地點。

![關鍵字搜尋](https://i.imgur.com/example.png)  <!-- 預留圖片位置 -->

### 使用「綠活標籤」篩選

我們提供了一系列的「綠活標籤」按鈕（例如：#全素/蔬食店, #環保餐具）。

- **點擊標籤**：地圖將只顯示包含該標籤的地點。
- **清除篩選**：點擊「所有地點」按鈕，即可清除標籤篩選，顯示所有地點。

## 3. 查看地點詳情

地圖上的每一個圖釘都代表一個綠活地點。

- **點擊圖釘**：點擊您感興趣的圖釘後，螢幕下方會滑出該地點的「資訊面板」。
- **查看資訊**：面板中會顯示該地點的詳細資訊，包括：
  - 名稱與地址
  - 照片預覽
  - 詳細描述
- **關閉面板**：點擊面板右上角的 `×` 按鈕即可關閉，回到全螢幕地圖。
